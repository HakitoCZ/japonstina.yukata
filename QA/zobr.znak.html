Text pod tímto odstavcem je způsob, jakým váš prohlížeč zobrazuje japonské písmo.

すべての人間は、生まれながらにして自由であり、
かつ、尊厳と権利と について平等である。
人間は、理性と良心とを授けられており、
互いに同胞の精神をもって行動しなければならない。

Můžete to porovnat s obrázkem níže.

Windows XP, 2000, Vista, 7 a 8 by měly mít Východo-asijské jazyky v základní instalaci. Pokud tak není, můžete je doinstalovat z Windows CD-ROM. V Ovládacích panelech je Místní a Jazykové nastavení > Jazyky. Minimální instalace zabere 230MB na disku.

Všechny Mac OS X od verze 10.4+ obsahují východo-asijské jazyky nativně.

GNOME, KDE i většina Debian-based GNU/Linux obsahují východo-asijské sady znaků nativně. Pokud tomu tak není, můžete je doinstalovat.

sudo apt-get install ttf-sazanami-mincho ttf-sazanami-gothic

Fedora
yum install fonts-japanese

Gentoo
# emerge arphicfonts

FreeBSD
# pkg_add -r ja-kochi-ttfonts
